# ergast

Для примера возьмем API с историей гоночных заездов http://ergast.com/mrd/
<br/>
Использовать JSON формат работы с API<br/>
Сделать дата-таблицы гонщиков с постраничной навигацией. Пользоваться необходимо навигацией из API, а не грузить все данные сразу.<br/>
Реализовать возможность перехода к подробной информации о гонщике по клику на его имени.<br/>
Реализовать таблицу заездов при клике по ссылке из таблицы с гонщиками.<br/>
Шаблон проекта должен быть React Native CLI (не использовать Expo CLI) - без интеграция TypeScript (tsx). Опыт использования и знания TypeScript приветствуется, но в данном задании нам нужно проверить навыки именно  в Java Script.<br/>
Использовать JS-библиотеки в обязательном порядке:<br/>
– ui - стандартные компоненты react-native (этого будет достаточно)<br/>
– react-navigation<br/>
– axios<br/>
– react-redux (в чистом виде без toolkit и т.п)<br/>
– redux-thunk или redux-saga на выбор<br/>
– будет плюсом redux-persist<br/>
